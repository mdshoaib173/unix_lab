WRITE A SHELL SCRIPT USING SELECT COMMANDS TO DISPLAY THE MENU:


SCRIPT:
PS3='SELECT NAMES : '
names='shoaib iftekar abid quit'
select name in $names
do
        if [ $name == 'quit' ]
        then
                break
        fi
        echo hello $name

done


OUTPUT:
